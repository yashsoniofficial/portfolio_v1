A self-balancing two-wheeled robot built as a hands-on exploration of control stability in robotics. The system utilized an Arduino Uno, L298N motor driver, DC gear motors, an MPU6050 IMU, and manually tuned PID control to maintain balance using only sensor feedback. The project served as a practical exercise in feedback systems, real-time control, and PID tuning on resource-constrained hardware, demonstrating the fundamentals of robotics control without reliance on advanced stacks.
